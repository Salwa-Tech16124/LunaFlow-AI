## 🌸 About LunaFlow AI
LunaFlow AI is a premium, beautifully designed wellness application built for modern women. Inspired by luxury apps like Flo and Clue, it seamlessly integrates cutting-edge AI predictions with an elegant, mobile-first Sakura-themed UI. 

Whether tracking menstrual cycles, logging daily hydration, or generating intelligent self-care insights, LunaFlow provides a fully private and responsive experience on both Desktop browsers and Android mobile devices.

---

## ✨ Features

* 🩸 **Intelligent Cycle Tracking**: Log your period and let the prediction engine calculate your next cycle, ovulation window, and fertile days.
* 🧘‍♀️ **Wellness & Symptom Logging**: Track mood swings, energy levels, cramps, sleep, and daily hydration.
* 🤖 **AI Health Assistant**: Ask Luna anything! An integrated generative AI assistant designed to provide helpful, personalized wellness advice.
* 🔔 **Smart Notification Center**: Fully customizable Period, Ovulation, and Medicine reminders built on a Firebase Push Notification backend architecture.
* 💅 **Luxury Glassmorphism UI**: A visually stunning Light/Dark mode featuring smooth CSS animations, floating Sakura petals, and fully responsive sliding cards.
* 📥 **Secure Data Export**: Instantly download and export your highly-encrypted health data to CSV format.

---

## 🛠️ Technology Stack

LunaFlow AI is engineered for robust performance, absolute security, and immediate cross-platform deployment.

* **Frontend**: Streamlit, Custom CSS3 Media Queries, Plotly (Interactive Responsive Charts)
* **Backend**: Python 3.12, Pandas (Data Processing & AI Analysis)
* **Database**: SQLite3 (Multi-tenant, isolated secure architecture)
* **Security**: SHA-256 Hash Encryption, rigorous user-session management
* **Mobile Ecosystem**: Android WebView Architecture, Firebase Cloud Messaging (FCM)

## 📱 Mobile-First & Android Ready
LunaFlow AI was built from the ground up specifically for mobile devices. 

It features a fluid, native-feeling UI that scales perfectly down to `360px` phone screens without horizontal scrolling, clipping, or ghost whitespace. The application is officially packaged for **Android Studio WebView**, allowing it to be wrapped instantly into a pristine `.apk` or `.aab` file for the Google Play Store.

## 🚀 Quick Start (Local Deployment)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Salwa-Tech16124/LunaFlow-AI.git
   cd LunaFlow-AI
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Launch the application:**
   ```bash
   streamlit run app.py
   ```
   *The secure local SQLite database (`lunaflow.db`) and all required tables will automatically generate and self-heal upon first launch.*

---

## 🔒 Privacy & Security First
LunaFlow AI deeply respects user data. All cycle histories, symptom logs, personal AI chat histories, and wellness tracking data are strictly sandboxed per user account. We utilize automatic zero-downtime schema migrations to fiercely protect data integrity during updates.
